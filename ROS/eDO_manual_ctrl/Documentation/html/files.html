 edo_manual_ctrl.cppROS node allows for e.DO to be controlled via Linux terminal. This file and the classes that it uses were created to eliminate the e.DO's dependency on its Android tablet application and allow it to be controlled via Linux terminal. This node has all of the basic functionality provided by the Android application and can be helpful in understanding how the e.DO Robot can be controlled by any ROS compatible program
